# Smart Local Search

> [🇺🇸 English](README.md) | [🇷🇺 Русский](README.ru.md)

Локальный AI-поисковик для персональных компьютеров с приоритетом конфиденциальности.
Находите файлы по смыслу, содержимому и контексту — вся обработка происходит исключительно на вашем компьютере.

## Что умеет

Smart Local Search индексирует ваши файлы и позволяет искать по ним на естественном языке:

- **«найди документ, где штраф за просрочку 0.5%»** — поиск по содержимому документов
- **«найди таблицу, где считал маркетинговый бюджет»** — семантический поиск по смыслу
- **«найди файл с кодом, где логика переподключения websocket»** — поиск по коду
- **«найди фото с текстом»** — поиск текста на изображениях через OCR
- **«найди картинки с закатом на пляже»** — визуальный поиск по описанию изображения
- **«найди папку с дизайн-ассетами»** — поиск по папкам и путям

## Гарантия конфиденциальности

**Вся индексация и поиск происходят локально. Ваши файлы никогда не покидают ваш компьютер.**

- 100% локальная обработка — без облачных API, без серверов
- Полностью работает офлайн после начальной настройки
- Без регистрации, без телеметрии, без отслеживания
- Архитектура делает случайную утечку данных невозможной

Сеть используется только опционально — для загрузки AI-моделей с Hugging Face и проверки обновлений на GitHub.

## Возможности

- **Индексация множества форматов** — Text, PDF, DOCX, Excel (XLSX/XLS), CSV и 20+ языков программирования
- **Полнотекстовый поиск** — SQLite FTS5 для мгновенного поиска по ключевым словам
- **Семантический поиск** — поиск по смыслу с помощью локальных эмбеддингов (multilingual-e5-small)
- **Поиск по коду** — разбивка кода с учётом языка, определение функций/классов, навигация по строкам
- **OCR-поиск по изображениям** — извлечение и поиск текста на изображениях через локальный Tesseract OCR
- **Визуальный поиск изображений** — поиск изображений по описанию через локальную модель CLIP
- **Нечёткий поиск** — находит файлы даже при неточных или ошибочных запросах
- **Умное ранжирование** — результаты ранжируются по текстовому совпадению + семантической близости + актуальности
- **Подсветка совпадений** — фрагменты текста с подсвеченными совпадениями в контексте
- **Клавиатурное управление** — полная навигация с клавиатуры (стрелки, Enter, Escape)
- **Действия с файлами** — открыть файл, открыть на строке (код), показать в папке
- **Инкрементальная индексация** — переиндексирует только изменённые файлы по хешу содержимого
- **Загрузка моделей в приложении** — скачивание AI-моделей прямо из настроек с отображением прогресса
- **Тёмный интерфейс** — чистый, современный дизайн на Tailwind CSS

## Поддерживаемые форматы файлов

| Категория | Расширения |
|-----------|-----------|
| Текст | `.txt`, `.md` |
| Документы | `.pdf`, `.docx`, `.csv`, `.xlsx`, `.xls` |
| Код | `.js`, `.ts`, `.jsx`, `.tsx`, `.py`, `.rs`, `.go`, `.java`, `.cs`, `.cpp`, `.c`, `.h`, `.hpp`, `.rb`, `.sh`, `.json`, `.html`, `.css`, `.yaml`, `.yml`, `.toml`, `.xml`, `.sql` |
| Изображения | `.jpg`, `.jpeg`, `.png`, `.webp` (извлечение текста OCR + визуальный поиск CLIP) |

## Технологический стек

| Компонент | Технология |
|-----------|-----------|
| Фреймворк | [Tauri 2](https://tauri.app/) |
| Фронтенд | React, TypeScript, Tailwind CSS, Vite |
| Бэкенд | Rust |
| База данных | SQLite с FTS5 |
| Текстовые эмбеддинги | [multilingual-e5-small](https://huggingface.co/intfloat/multilingual-e5-small) (ONNX, 384-dim) |
| Эмбеддинги изображений | [CLIP ViT-B/32](https://huggingface.co/Marqo/onnx-open_clip-ViT-B-32) (ONNX, 512-dim) |
| OCR | [Tesseract](https://github.com/tesseract-ocr/tesseract) (системная зависимость) |

## Установка

### Скачать

Скачайте последнюю версию со страницы [GitHub Releases](https://github.com/nicholasgriffintn/smart-local-search/releases):

- **macOS**: установщик `.dmg` (Apple Silicon)
- **Windows**: установщик `.msi` или `.exe`

### Первый запуск

1. Откройте приложение и перейдите в **Настройки**
2. Скачайте модель **multilingual-e5-small** для семантического текстового поиска
3. *(Опционально)* Скачайте модель **CLIP ViT-B/32** для визуального поиска изображений
4. Добавьте папки для индексации
5. Начинайте искать

Все модели загружаются прямо в приложении — команды в терминале не нужны.

### Опционально: OCR-поиск по изображениям

Для поиска текста внутри изображений установите Tesseract OCR:

```bash
# macOS
brew install tesseract

# Ubuntu / Debian
sudo apt install tesseract-ocr

# Windows — скачайте установщик:
# https://github.com/UB-Mannheim/tesseract/wiki
```

Без Tesseract приложение работает нормально — функции OCR просто пропускаются.

## Сборка из исходников

### Требования

- [Rust](https://rustup.rs/) (1.77.2+)
- [Node.js](https://nodejs.org/) (18+)
- [Зависимости Tauri](https://tauri.app/start/prerequisites/)

### Сборка

```bash
git clone https://github.com/nicholasgriffintn/smart-local-search.git
cd smart-local-search/desktop

npm install

# Режим разработки (горячая перезагрузка)
npm run tauri dev

# Продакшен-сборка
npm run tauri build
```

Результат сборки:

- **macOS**: `src-tauri/target/release/bundle/dmg/`
- **Windows**: `src-tauri/target/release/bundle/nsis/`
